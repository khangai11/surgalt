## Функц руу array дамжуулах жишээнүүд

```c
//size хэмжээтэй A хүснэгтийг гараас унших
void Ain(int A[], int size){
  for(int i=0;i<size;i++){
    cin>>A[i];
  }
}
```
```c
//size хэмжээтэй A хүснэгтийн элементүүдийг тэглэх
void Azero(int A[], int size){
  for(int i=0;i<size;i++){
    A[i] = 0;
  }
}
```
