Бодлогын тайлбар:
N ширхэг бүхэл тоонууд болох A_1, A_2, ..., A_N өгөгдөнө. Мөн B_i = A_i × A_(i+1) (1 ≤ i ≤ N-1) гэж тодорхойлогдоно.

B_1, B_2, ..., B_(N-1) утгуудыг энэ дарааллаар, хооронд нь зайтайгаар хэвлэнэ.

Хязгаарлалт:
2 ≤ N ≤ 100
1 ≤ A_i ≤ 100
